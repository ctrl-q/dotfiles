# dotfiles

Personal configuration files for a Linux desktop environment.

## Contents

| File/Directory | Description |
|---|---|
| `.bashrc` / `.bash_profile` / `.bash_aliases` | Bash shell configuration and aliases |
| `.vimrc` | Vim editor configuration |
| `.gitconfig` | Git global configuration |
| `.sqliterc` | SQLite CLI configuration |
| `.terraformrc` | Terraform CLI configuration |
| `.xinitrc` | X session startup script |
| `.config/openbox/` | Openbox window manager configuration |
| `.config/yazi/` | Yazi file manager configuration and plugins |
| `.config/neofetch/` | Neofetch system info display configuration |
| `.config/eget/` | eget binary downloader configuration |
| `.config/worktrunk/` | Worktrunk configuration |

## Usage

Clone and symlink or copy the files to your home directory, e.g. using [GNU Stow](https://www.gnu.org/software/stow/) or a bare git repository approach.
